## 📦 安装方法

### 前置条件
- Node.js ≥ 22
- 已安装 ClawHub CLI

### 安装命令
```bash
# 单个安装
clawhub install <skill-name>

# 批量安装（推荐）
clawhub install capability-evolver self-improving-agent github wacli summarize
```

### 验证安装
```bash
clawhub list  # 查看已安装技能
```
