## 🚀 使用方式

```bash
# 使用 Midnight（默认）
python skills/ai_daily_v2.py

# 使用 Sunset
python skills/ai_daily_v2.py --scheme sunset

# 使用 Ocean
python skills/ai_daily_v2.py --scheme ocean

# 使用 Purple
python skills/ai_daily_v2.py --scheme purple
```

## 💡 自定义配色

编辑 `ai_daily_v2.py` 中的 `COLOR_SCHEMES` 字典：

```python
COLOR_SCHEMES = {
    'my_custom': {
        'bg_start': (R, G, B),
        'bg_end': (R, G, B),
        'accent': (R, G, B),
        'text_dark': (R, G, B),
        'text_light': (R, G, B),
        'card_bg': (R, G, B, Alpha),
    }
}
```
