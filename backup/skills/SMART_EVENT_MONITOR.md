# 🚨 Smart Event Monitor - 智能事件监控

**"捕捉每一个交易机会"**

智能事件驱动交易监控技能，专为捕捉类似"OpenClaw 发言→MiniMax 升浪"的机会而设计。

---

## 🎯 功能特性

### 1️⃣ 多源监控
- ✅ 财联社电报（7×24 小时快讯）
- 🔄 华尔街见闻（待接入）
- 🔄 推特关键人物（待接入）
- 🔄 公司公告（待接入）

### 2️⃣ 智能关联
```
事件 → 相关股票
人物 → 关联公司
行业 → 产业链股票
地缘 → 影响板块
```

### 3️⃣ 重要性评分
| 等级 | 关键词 | 示例 |
|------|--------|------|
| 🔴 极高 | 爆炸、袭击、战争、IPO、并购 | 地缘冲突、重大并购 |
| 🟡 高 | 原油、美股、港股、发布、合作 | 市场动态、公司合作 |
| 🟢 中 | 会议、计划、预计 | 常规新闻 |

### 4️⃣ 飞书推送
- 实时推送到指定群聊
- 重要性标记（🔴🟡🟢）
- 股票关联信息
- 去重防骚扰

## 🔧 配置

### 添加新股票
编辑 `STOCK_MAPPING` 字典：
```python
"公司名称": {
    "stock": "股票代码",
    "market": "市场",
    "keywords": ["关键词 1", "关键词 2"]
},
```

### 调整重要性规则
编辑 `IMPORTANCE_RULES` 字典：
```python
"🔴 极高": ["关键词 1", "关键词 2"],
"🟡 高": ["关键词 3", "关键词 4"],
```

### 修改监控分类
编辑 `MONITOR_CATEGORIES` 字典。
